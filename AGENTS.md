# Codex Harness Starter

이 저장소는 빈 프로젝트 디렉토리에서 바로 사용할 수 있는 한국어 기반 에이전트 팀 하네스를 만드는 Codex 플러그인입니다.

## 기본 철학

- 하네스의 기본 산출물은 문서 묶음이 아니라 에이전트 팀입니다.
- 역할 스킬보다 역할 에이전트 생성을 먼저 중요하게 다룹니다.
- 모든 하네스에는 가능한 한 검증 역할이나 검증 단계가 포함되어야 합니다.
- 결과물은 현재 프로젝트 안에서 바로 사용할 수 있어야 합니다.

## 기본 구조

```text
AGENTS.md
.agents/
  skills/
.codex/
  agents/
```

## 사용 방식

사용자는 빈 프로젝트 디렉토리에서 `harness-creator`를 호출해 도메인에 맞는 에이전트 팀 하네스를 생성합니다.

## 기본 산출물

- 루트 `AGENTS.md`: 구조, 사용법, 산출물 정의
- `.codex/agents/*.toml`: 역할별 에이전트
- `.agents/skills/<team-skill>/SKILL.md`: 팀 오케스트레이션 스킬
- 필요 시 검증 규칙 또는 검증 전용 역할 정의
