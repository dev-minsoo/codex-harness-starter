# Pitch Deck Framework

10장 안팎의 투자자용 피치덱 구조를 설계합니다.

## 기본 슬라이드

- 문제
- 해결책
- 시장 규모
- 제품
- 비즈니스 모델
- 경쟁 우위
- 성장 전략
- 트랙션 또는 검증 상태
- 팀
- 투자 요청

## 규칙

- 스토리는 문제에서 투자 요청까지 자연스럽게 이어져야 합니다.
- 주장마다 근거 또는 현재 검증 상태를 표시합니다.
- 공격적인 예상은 별도 가정으로 구분합니다.
